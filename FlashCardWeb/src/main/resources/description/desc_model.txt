.	class construction 資料夾描畫 重要 class 的結構,使用 FreeMind 軟體開啟

.	word 的 id 以e開頭,表系統的預設id,ex: e1234
	自訂的詞彙 以 其它開頭並以底線連接序號,ex: cust_1234
	
.	user 的 authority 格式為 xxx,yyy ,ex:READER,USER
	其定義在 Authority 的 subclass ,ex: AdminAuthority
	UserAuthority:對word 的使用權限
	AdminAuthority:對會員帳號的管理權限

.	model/dao 的 user 指 使用者帳號資料; word 是 詞彙相關資料

.	beans 的property 的 element 有些 props ,key 用在 controller, 
	value 則用在 .jsp 的 EL 語法,例如: MemberController 和 member.jsp
